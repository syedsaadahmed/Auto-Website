Intro

Perfect Gear is a template well suited for various businesses related to automotive industry: auto repair, auto services, auto maintenance, car wash, auto parts store, auto paint shop, etc.

We have researched many different auto repair and auto shop sites, and made a thorough analysis of these sites taking into account views, needs and comments from motorists. The exclusive style and user-friendly interface of this template meets the needs of both businesses and their customers.

Template features

  • 9 HTML Pages
  • Based on Bootstrap
  • Responsive design: mobile, tablet & desktop layouts are supported
  • Free Fonts (Google Fonts) & Icons used
  • Easy to customize
  • And more

Based on Bootstrap

This template is based on Twitter Bootstrap and Bootstrap Grid system is mostly used for laying out content on the page and inside sections.

Note: Bootstrap files are already included in this template, thus you don't need to download or configure anything for theme to work properly.

List of HTML files

  • index.html - Home page
  • index-no-slider.html - Home page without slider
  • services.html - Detailed list of services
  • about.html - Information about the company
  • shop.html - Parts shop
  • blog.html - Blog archive page
  • blog-sidebar-left.html - Blog archive page with a sidebar on the left
  • post.html - Single blog post page
  • contact.html - Contacts page

Folder structure

Apart from HTML files there are also 3 folders:

  • css - Contains style.css - the main theme stylesheet. This folder also contains third party stylesheets and their minified versions.
  • js - Contains script.js - the main theme script file. This folder also contains third party JavaScript files and their minified versions.
  • fonts - Contains necessary font files.

HTML structure

Header

The header of each page is enclosed inside the header tag.

Sections

The content of the page is subdivided into sections. Each section is enclosed inside it's own section tag.

Sections are independent blocks, so you should be able to move one section around and modify it without affecting other sections.

Footer

Each page also has a footer enclosed inside the footer tag.
